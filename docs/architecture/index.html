lash/system

lash is a Rust workspace for embeddable coding agents. The core is a layered agent runtime — a sans-IO turn machine wrapped by an async host runtime, with mode plugins, provider adapters, dynamic tools, durable session graph storage, embedding APIs, benchmark runners, and a CLI/TUI host on top.
